# 🐦 Bird Species Observation Analysis

## 📊 Project Overview
This project analyzes bird observation data across forest and grassland ecosystems to understand biodiversity patterns, species distribution, and environmental impact.

The analysis includes data cleaning, exploratory data analysis (EDA), SQL querying, and dashboard visualization using Power BI.

---

## 🎯 Objectives
- Analyze bird species distribution across habitats  
- Identify seasonal trends in bird observations  
- Study environmental factors such as temperature and humidity  
- Evaluate observation methods  
- Identify at-risk species for conservation  

---

## 🛠 Tools & Technologies Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)  
- SQL  
- Power BI  
- Jupyter Notebook  

---

## 📂 Project Workflow
1. Data collection from multiple Excel sheets  
2. Data cleaning and preprocessing  
3. Feature engineering (Year, Month extraction)  
4. Exploratory Data Analysis (EDA)  
5. SQL analysis for insights  
6. Power BI dashboard creation  

---

## 📈 Key Insights
- Bird observations vary significantly across months  
- Habitat type influences species diversity  
- Environmental factors affect bird activity  
- Observation methods such as singing and calling dominate  
- A small percentage of species are classified as at-risk  

---

## 📊 Power BI Dashboard
The dashboard provides interactive visualizations including:
- Monthly observation trends  
- Top bird species  
- Habitat comparison  
- Environmental analysis  
- Observation method distribution  
- At-risk species analysis
